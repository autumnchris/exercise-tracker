View the source code for this project.

Project Usage

Create a User

Create a username in the form below to get a corresponding user ID that you can use to add exercises for that registered user.

Add Exercise

Add an exercise in the form below for a registered user.

Get a List of Users

To get a JSON array of all the registered users, go to the path /api/exercise/users.

Get Exercise Data for a User

To get JSON data for a specific user, go to the path /api/exercise/log and then add a parameter of the user's ID. For example, if I wanted to get the data for a user whose ID is 5bea63e40f02c48aab5f0e38, I would type /api/exercise/log/5bea63e40f02c48aab5f0e38.

I can also filter the list of exercise logs that get returned. I can limit the quantity of logs by adding a query input to the path such as /api/exercise/log/5bea63e40f02c48aab5f0e38?limit=2. I can also return logs within a specific date range such as /api/exercise/log/5bea63e40f02c48aab5f0e38?from=2013-10-31&to=2019-12-25. I can even both limit the quantity and select a date range such as /api/exercise/log/5bea63e40f02c48aab5f0e38?from=2013-10-31&to=2019-12-25&limit=2.
