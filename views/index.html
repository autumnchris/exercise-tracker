My project for the Exercise Tracker challenge as part of the curriculum for the Apis and Microservices Certification on Free Code Camp.

View the source code for this project.

Project Usage

Create a User

Create a username in the form below to get a corresponding user ID that you can use to add exercises for that registered user.

Add Exercise

Add an exercise in the form below for a registered user.

Note: To submit the current date, leave the Date input field blank.

Get a List of Users

To get a JSON array of all the registered users, go to the path /api/exercise/users.
